# Changelog

All notable changes to Icon Map will be documented in this file.

## [1.0.0] - 2025-01-15

### Added
- Initial release
- Grid view of all system icons
- Search functionality (by name and category)
- Copy icon name to clipboard
- Category-based organization
- Context menu with copy SVG option
- Modular component architecture
- Scoped CSS styling
- Keyboard shortcuts support

### Technical
- Modular architecture with SearchBar and IconGrid components
- Integration with Icons.js utility
- Clipboard API integration
- Event-driven component communication
