## Usage

1. Launch Icon Map from the Start Menu
2. Browse icons in the grid or use search
3. Click on an icon to copy its name
4. Right-click for more options (copy SVG code)

## Keyboard Shortcuts

- `Ctrl+F` - Focus search
- `Escape` - Clear search
- `Ctrl+C` - Copy selected icon

## Permissions

- **clipboard** - Required to copy icon names and SVG code
